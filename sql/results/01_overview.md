# Overview

**Проект:** Анализ оттока клиентов телеком-компании (IBM Sample Data Sets)

**Цель:** Выявить ключевые факторы оттока клиентов и подготовить данные для дальнейшего анализа, кластеризации и обучения модели.

**Данные:** 7043 записи, 21 признак.

**Ключевые метрики:**
| Показатель             | Значение   |
|------------------------|------------|
| Всего клиентов         | 7043       |
| Ушедших клиентов       | 1869       |
| Churn Rate             | 26.54%     |
